# VideoBelajar - Platform Edukasi

Platform pembelajaran video interaktif untuk meningkatkan skill dengan ribuan kursus berkualitas dari instruktur berpengalaman.

## Struktur Proyek

```
mission3/
├── home.html          # Halaman utama dengan daftar kursus
├── login.html         # Halaman login pengguna
├── register.html      # Halaman registrasi akun baru
├── style.css          # Styling untuk halaman home
├── login.css          # Styling untuk halaman login
├── register.css       # Styling untuk halaman register
├── img/               # Folder untuk gambar lokal
└── test.txt           # File teks
```

## Fitur

- Landing page dengan koleksi kursus video
- Halaman login dengan opsi login Google
- Form registrasi dengan validasi password
- Navigasi responsif menggunakan Bootstrap 5
- Toggle visibility untuk password
- Desain modern dengan Bootstrap Icons

## Teknologi

- HTML5
- CSS3
- Bootstrap 5.3.0
- Bootstrap Icons 1.11.3

## Cara Menjalankan

Buka `home.html` di browser untuk melihat landing page kursus. (if you want to open via VCS download live server extension)
